Detailed Description

Archivo de cabecera del archivo fuente oneWireSensor.c.

Manejo de sensor tipo OneWire.

Note
Sensor utilizado Maxim ds18b20. Se considera al sensor conectado con una resistena de pullup de 4,7K.
