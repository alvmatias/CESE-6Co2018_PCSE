int32_t  TCPUDPPort,
uint8_t  TCPUDPConnectionId 
)

Envio de data via TCP/UDP.

Parameters
data: Puntero al buffer que almacena los datos a enviar.
transportLayer: Capa de transporte a utilizar para enviar los datos.
TCPUDPIp: IP del servidor al cual enviar los datos.
TCPUDPPort: Puerto TCP/UDP del servidor con el cual establecer la conexion.
TCPUDPConnectionId: Interfaz TCP/UDP a utilizar(0 - 4).
Returns
ESP8266_NOT_WORKING Si la inicializacion NO fue correcta, ESP8266_WORKING caso contrario.
Note
La interfaz TCP/UDP puede ser distinto de 0 solo si TCPUDPConnectionType fue seteado como MULTIPLE_CONNECTION.

error : Estado de funcionamiento del modulo Wifi
